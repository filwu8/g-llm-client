# G-LLM License Policy

This policy explains which license applies to each G-LLM Desktop Client
release. If this summary conflicts with the license included in a release,
the license included in that release controls.

## Version Boundary

- V1.0.10 and earlier: `AGPL-3.0-only`, as preserved in each Git tag.
- V1.1.0: `BUSL-1.1`, with the Additional Use Grant and Change Date stated in
  [LICENSE](./LICENSE).
- Later versions: the license and Change Date included in that release.

Changing the license for a later release does not revoke rights already
granted for an earlier release.

## Free Use of V1.1.0

V1.1.0 may be used without charge by individuals and for an organization's
own internal business operations. Source inspection, modification, and
non-production redistribution are allowed under the Business Source License
1.1.

White-label, OEM, resale, competing distribution, third-party hosted or
managed service use, and other Production Use outside the Additional Use
Grant require a separate written commercial license.

## Four-Year Conversion

V1.1.0 converts to `AGPL-3.0-only` on 2030-07-14. Release managers must set a
specific Change Date for every later release and preserve the license file in
the corresponding Git tag and source archive.

## Included and Excluded Rights

The Licensed Work includes first-party source code and documentation in this
repository unless a file states a different license or is identified as a
third-party component.

Third-party components remain under their own licenses. See
[THIRD_PARTY_NOTICES.md](./THIRD_PARTY_NOTICES.md).

The source license does not grant trademark rights, hosted service access,
API credits, private datasets, private infrastructure, or commercial brand
authorization.

## Contributions

External contributions are accepted only after the contributor has completed
the contribution process described in [CONTRIBUTING.md](./CONTRIBUTING.md).
GPROPHET LIMITED must retain sufficient rights to distribute accepted
contributions under both the source-available and commercial licensing
models.
