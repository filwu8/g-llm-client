# 无极界 G-LLM V1.0 发布说明

版本号：`1.0.0`

## 版本定位

V1.0 是无极界 G-LLM 桌面客户端的首个正式发布版本，核心目标是让用户不依赖第三方客户端，也能通过自有品牌客户端完成助手选择、模型配置和多轮对话。

## 核心能力

- 多助手入口：内置通用、文档、合同、代码、经营分析、学习、建筑工程等场景助手。
- 助手中心：支持搜索预置助手、创建自定义助手、AI 生成助手配置。
- 多会话聊天：左侧选择助手，右侧管理当前助手会话。
- 模型与供应商：默认 G-LLM 网关，支持新增 OpenAI-compatible、OpenAI、Gemini、DeepSeek、通义千问、Kimi、智谱、火山方舟、SiliconFlow、OpenRouter、Ollama、LM Studio 等模板。
- 会话级模型：助手可设置默认模型，每个会话可单独选择模型。
- 附件能力：支持文件、图片、剪贴板粘贴附件和系统截图入口。
- 本地知识库：支持将消息保存到当前助手知识库，并在对话中手动引用。
- 长期记忆：支持当前助手范围内的长期记忆。
- 联网搜索：支持将搜索资料注入提问上下文。
- Markdown 渲染：支持列表、表格、代码块等常见 Markdown 输出。
- Token 展示：显示单条消息和会话总词元，区分输入/输出。
- 匿名统计：默认开启，仅上报匿名元数据，不采集聊天内容、API Key、文件内容和知识库内容，用户可在设置中关闭。

## 发布前检查

- `pnpm build` 通过。
- 默认供应商为 `https://llm.gprophet.com/v1`。
- 首次使用需要用户自行填写 API Key。
- 匿名统计默认开启，用户可在设置中关闭。
- `package.json` 版本为 `1.0.0`。
- Windows 安装包可通过 `pnpm package:win` 构建。

## Windows 安装包

- 安装包：`dist/G-LLM-Setup-1.0.0-x64.exe`
- 安装器类型：NSIS 向导式安装包。
- 安装范围：默认当前用户安装，不要求管理员权限。
- 安装目录：用户可在安装向导中修改。
- 快捷方式：创建桌面快捷方式和开始菜单快捷方式。
- 卸载策略：默认不删除用户数据，便于卸载重装或升级后保留配置。
- 数字签名：当前 V1.0 安装包未配置代码签名证书，公开分发前建议补充签名。

## Windows 数据规划

- 用户数据根目录：`%APPDATA%/G-LLM`
- 主配置文件：`%APPDATA%/G-LLM/g-llm-client.json`
- 当前存储内容：供应商配置、API Key、助手、会话、轻量知识库、长期记忆、工具配置、匿名安装 ID 和用户设置。
- 不放入安装目录：用户配置和会话数据不写入 `Program Files` 或应用安装目录，避免升级覆盖和权限问题。
- 升级策略：后续版本继续沿用 `%APPDATA%/G-LLM`，需要结构变化时通过应用内迁移逻辑处理。
